# timer
Rust implementation of a Pomodoro timer.

```text
timer 1.0
A simple Pomodoro timer.

USAGE:
    timer [OPTIONS]

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -c, --color <COLOR>      Sets the duration of the timer (blue, cyan, green, red, white, yellow)
    -f, --font <SIZE>        Sets the font size (5, 6, 7, 20)
    -t, --time <DURATION>    Sets the duration of the timer (35s, 2m, 15m+30, ...)
```
